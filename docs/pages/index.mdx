# Introduction

**uno-shadcn-ui** is based on [**shadcn/ui**](https://ui.shadcn.com/), but replace tailwindcss with [**unocss**](https://unocss.dev/).

## BaseOn

- vite
- radix-ui
- unocss
  - unocss-preset-icon
- shadcn-ui

## Manual Installation

- Install unocss
- Copy `preset.shadcn.ts` in this project, then add it into `uno.config.ts`
- Copy code into yout project or `pnpm add uno-ui:github:fisand/uno-shadcn-ui`
