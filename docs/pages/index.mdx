# Introduction

**uno-shadcn-ui** is based on [**shadcn/ui**](https://ui.shadcn.com/), but replace tailwindcss with [**unocss**](https://unocss.dev/).

## Base on

- vite
- radix-ui
- unocss
  - unocss-preset-icon
- shadcn-ui

## Manual Installation

> Use the components codes in this project is **unnecessary**. Just need `unocss` preset.

- [x] install `unocss`
- [x] touch `uno.config.ts`. copy `preset.shadcn.ts` into your project from [here](https://github.com/fisand/uno-shadcn-ui/blob/main/preset.shadcn.ts), then add it into `uno.config.ts`.
