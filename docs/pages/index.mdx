# Introduction

**uno-shadcn-ui** is based on [**shadcn/ui**](https://ui.shadcn.com/docs/components/accordion), but replace tailwindcss with [**unocss**](https://unocss.dev/).

## BaseOn

- vite
- radix-ui
- unocss
  - unocss-preset-icon
- shadcn-ui

## Manual Installation

- Install unocss
- Copy `preset.shadcn.ts` in this project, then add it into `uno.config.ts`
