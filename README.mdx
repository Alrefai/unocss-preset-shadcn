# shadcn-ui

shadcn-ui but **unocss**

## quick start
> Use the components codes in this project is **unnecessary**. Just need `unocss` preset.

- [x] install `unocss`
- [x] touch `uno.config.ts`. copy `preset.shadcn.ts` in this project, then add it into `uno.config.ts`.
